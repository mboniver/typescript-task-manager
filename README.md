# TypeScript Task Manager

Ein einfacher Task-Manager zur Verwaltung von ToDo-Listen, entwickelt mit TypeScript.

Ziel des Projekts ist es, meine Kenntnisse in TypeScript praktisch anzuwenden und grundlegende Konzepte der Softwareentwicklung zu vertiefen.

## Funktionen

- Aufgaben erstellen
- Aufgaben anzeigen
- Aufgaben löschen
- Aufgaben als erledigt markieren

## Eingesetzte Mittel

- TypeScript
- JavaScript
- HTML
- CSS
- Git
